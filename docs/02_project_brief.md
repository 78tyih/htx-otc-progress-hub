# HTX OTC BD Progress Hub 项目说明

> 文档版本：v1.0 ｜ 更新日期：2026-07-21 ｜ 维护人：Sera

## 1. 项目背景

Sera 当前处于 OTC USD 大宗业务恢复与 BD Pipeline 建设阶段，需要围绕 PIP 目标建立一个可视化进度看板。

目前已完成两项核心工作：

1. OTC USD 大宗交易设计交付包已整理完成，准备提交设计团队；
2. TG 存量客户资料和客户信息已汇总，已制作成 CRM 表格。

接下来需要将这些工作沉淀为一个可视化网页，让思源哥可以随时查看项目进度。

## 2. 项目目标

建立一个轻量级静态网页，用于：

- 展示 PIP 目标完成进度；
- 展示设计团队交付进度；
- 展示客户 CRM Pipeline；
- 展示注册、KYC、首单转化状态；
- 展示渠道拓展状态；
- 展示阻塞事项和所需协作；
- 支持每周更新。

## 3. 目标用户

| 用户 | 使用场景 |
| --- | --- |
| Siyuan.C / 思源哥 | 查看 Sera 工作进度、判断是否需要协助 |
| Sera | 每周更新 Pipeline 和 PIP 进展 |
| 设计团队 | 查看设计交付包内容和当前排期 |
| BD / 销售协作方 | 查看客户转介和高价值客户推进状态 |

## 4. 核心业务链路

```text
业务资料整理 → 设计交付包提交 → 客户 CRM 建表 → 客户分级 → 注册 / KYC 推进 → 首单交易测试 → 渠道拓展 → 周报 / CRIB 复盘
```
